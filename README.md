# SOC Level 1 Analyst Course

Beginner to Advanced SOC L1 Analyst training course.

## Course Topics

- SOC fundamentals
- SOC team roles
- Alerts, events, and logs
- Alert triage
- Alert verdicts
- Investigation methodology
- Identity inventory
- Asset inventory
- Threat intelligence lookups
- Network diagrams
- Workbooks, playbooks, runbooks, and workflows
- Alert reporting
- Alert escalation
- SOC communication
- SOC metrics and objectives
- SLA, MTTD, MTTA, and MTTR
- Practical labs
- Capstone project

## Repository Structure

- `modules/` - Main course lessons
- `labs/` - Practical labs
- `datasets/` - Sample logs, alerts, identity inventory, and asset inventory
- `templates/` - Report, escalation, workbook, playbook, and runbook templates
- `playbooks/` - SOC playbooks
- `runbooks/` - SOC runbooks
- `workbooks/` - SOC investigation workbooks
- `diagrams/` - SOC and network diagrams
- `quizzes/` - Module quizzes
- `solutions/` - Lab and quiz solutions
- `scripts/` - Helper scripts
- `references/` - Cheatsheets and useful references
